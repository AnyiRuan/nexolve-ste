# Nexolve 管理后台使用指南

## 概述

Nexolve 网站现在配备了一个简单易用的后台管理系统，专为没有编程经验的管理员设计。通过这个系统，您可以轻松管理网站内容和聊天机器人的知识库。

## 访问管理后台

### 1. 找到管理员入口

在网站的**任何页面底部**，您都可以找到一个小的 **"Admin"** 按钮（带锁图标）。点击这个按钮即可进入登录页面。

### 2. 登录

**默认密码**: `nexolve2025`

> ⚠️ **重要**: 首次部署后，强烈建议您修改默认密码！

#### 如何修改管理员密码

在服务器的环境变量中设置 `ADMIN_PASSWORD`：

```bash
# 方法 1: 在 .env.local 文件中添加
ADMIN_PASSWORD=您的新密码

# 方法 2: 如果使用 Vercel 等平台部署
# 在平台的环境变量设置中添加 ADMIN_PASSWORD
```

## 管理后台功能

登录后，您会看到两个主要管理模块：

### 📝 网页内容管理

在这个模块中，您可以编辑网站的核心内容：

1. **首页标题** - 网站主页的大标题
2. **首页副标题** - 主标题下方的描述文字
3. **关于我们文本** - 公司介绍内容
4. **联系邮箱** - 网站上显示的联系邮箱地址

**操作步骤**：
1. 点击 "网页内容管理" 标签
2. 在相应的输入框中修改文本
3. 点击 "保存更改" 按钮
4. 系统会显示保存状态（成功或失败）

> 💡 **提示**: 如果不满意修改，可以点击 "重置" 按钮恢复到上次保存的版本。

### 🤖 聊天机器人知识库

这个模块帮助您管理网站右下角的 AI 聊天助手。它包含两个部分：

#### 1. AI 助手角色设定（系统提示词）

这段文字定义了聊天机器人的"个性"和行为方式。

**示例**：
```
您是 Nexolve 的智能客服助手，专门帮助访客了解我们的服务。
请用友好、专业的语气回答问题，保持简洁明了。
```

**操作步骤**：
1. 在 "系统提示词" 文本框中输入或修改角色设定
2. 点击 "保存系统提示词" 按钮

#### 2. 常见问题（FAQ）

您可以添加常见的客户问题和标准答案，帮助 AI 更准确地回答访客。

**操作步骤**：

**添加新问题**：
1. 在 "添加新问题" 区域
2. 在 "问题" 框中输入问题（例如："你们的营业时间是？"）
3. 在 "答案" 框中输入详细答案
4. 点击 "添加问题" 按钮

**删除问题**：
- 点击每个问题右侧的垃圾桶图标即可删除

**保存更改**：
- 添加或删除问题后，点击底部的 "保存所有更改" 按钮

## 常见问题

### Q: 我修改的内容什么时候生效？

A: 点击 "保存" 按钮后，内容会立即保存到服务器。网站内容可能需要刷新页面才能看到；聊天机器人的知识库会在下次对话时生效。

### Q: 如果我忘记了密码怎么办？

A: 请联系网站技术管理员重置 `ADMIN_PASSWORD` 环境变量。

### Q: 能同时有多个管理员登录吗？

A: 可以，系统支持多个管理员同时登录和编辑。但请注意，后保存的内容会覆盖先前的修改。

### Q: 数据会丢失吗？

A: 所有数据都保存在服务器的 `data` 目录中，以 JSON 格式存储。建议定期备份这些文件。

### Q: 如何添加更多的常见问题？

A: 在 "常见问题" 区域，使用 "添加新问题" 表单，可以添加无限数量的问答对。

## 技术细节（供技术人员参考）

### 数据存储位置

- 网站内容: `/data/admin-content.json`
- 知识库: `/data/knowledge-base.json`

### API 端点

- 登录: `POST /api/admin/login`
- 内容管理: `GET/POST /api/admin/content`
- 知识库管理: `GET/POST /api/admin/knowledge-base`

### 权限验证

系统使用基于 localStorage 的简单 token 验证。对于生产环境，建议实施更强大的身份验证机制（如 JWT）。

## 安全建议

1. ✅ 修改默认密码
2. ✅ 使用强密码（至少 12 位，包含字母、数字和符号）
3. ✅ 定期备份 `data` 目录
4. ✅ 使用 HTTPS（部署在 Vercel 等平台时自动启用）
5. ✅ 不要在公共场所或不安全的网络中登录管理后台
