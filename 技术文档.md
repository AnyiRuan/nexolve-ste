# Nexolve 网站技术文档

## 快速参考

### 技术栈
- Next.js 14.2.5
- React 18.2.0
- Tailwind CSS 3.4.3
- next-intl 4.5.2（多语言）
- 部署：Vercel

### 支持语言
- 英文 (en) - 默认
- 中文 (zh)
- 西班牙语 (es)

## 常用操作

### 本地开发
```bash
npm install          # 安装依赖
npm run dev          # 启动开发服务器 (http://localhost:3000)
npm run build        # 构建生产版本
npm start            # 启动生产服务器
```

### 部署到生产环境
```bash
git add .
git commit -m "你的更改说明"
git push origin main
# Vercel 会自动部署
```

## 内容更新指南

### 1. 修改首页文字

**文件**: `messages/en.json`（英文）、`messages/zh.json`（中文）、`messages/es.json`（西班牙语）

```json
{
  "hero": {
    "title": "主标题",
    "subtitle": "副标题介绍",
    "ctaPrimary": "主按钮",
    "ctaSecondary": "次按钮"
  }
}
```

### 2. 修改服务内容

**位置**: `app/[locale]/services/服务名称/page.jsx`

例如修改 Website & Digital Growth 服务：
```
app/[locale]/services/website-digital-growth/page.jsx
```

### 3. 修改样式

**文件**: `tailwind.config.js` 或 `app/globals.css`

## 常见问题

**Q: 如何添加新页面？**
A: 在 `app/[locale]/` 下创建新文件夹，添加 `page.jsx` 文件。

**Q: 如何更改颜色主题？**
A: 修改 `tailwind.config.js` 中的颜色配置。

**Q: 如何添加新语言？**
A:
1. 在 `i18n/routing.js` 添加语言代码
2. 在 `messages/` 下创建新的 JSON 文件
3. 翻译所有内容

**Q: 构建失败怎么办？**
A:
1. 检查控制台错误信息
2. 确保 `npm install` 已运行
3. 删除 `.next` 文件夹后重新构建
